# How to install

* clone branch ini
* pastikan menginstall python antara versi 3.10 sampe 3.13
* membuat env yang disesuai dengan database :
  ```
  PGUSER= // nama user anda (biasanya untuk root 'postgres')
  PGHOST=localhost
  PGPASSWORD= // password postgres
  PGDATABASE= // nama database
  PGPORT=5432
  ```
